## Project Overview
This project analyzes unemployment trends in India using Python and real-world data.
It is submitted as part of the **AICTE Oasis Infobyte Internship (Data Science)**.

## Dataset
Source: https://www.kaggle.com/datasets/gokulrajkmv/unemployment-in-india

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Analysis Performed
- Data cleaning and preprocessing
- State-wise unemployment analysis
- Time series analysis
- COVID-19 impact on unemployment
- Urban vs Rural comparison

## How to Run
1. Download the dataset from Kaggle
2. Place `Unemployment in India.csv` in this folder
3. Open `Unemployment_Analysis.ipynb`
4. Run all cells
